## OIBSIP_WebDev_Task1 - Temperature Converter Website

## Objective
To build a simple and interactive **Temperature Converter web application** using HTML, CSS, and JavaScript.  
The project allows users to convert temperatures between **Celsius, Fahrenheit, and Kelvin** with proper input validation and real-time results.

## Tools Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- VS Code (Code Editor)  
- Chrome / Firefox (Browser)

## Features
- Input field for entering temperature value  
- Dropdown menu to select input unit (Celsius, Fahrenheit, Kelvin)  
- Convert button to perform temperature conversion  
- Displays converted values in all three units  
- Input validation to prevent invalid entries  
- Clean, modern, and responsive UI design  
- Smooth hover and transition effects for better user experience  

## Files Included
- Converter.html – Structure of the temperature converter interface  
- Converter.css – Styling and responsive layout design  
- Converter.js – Logic for temperature conversion and input validation  

## Steps Performed
1. Designed the user interface using HTML with input, dropdown, button, and result section.  
2. Styled the application using CSS with gradients, cards, and hover effects.  
3. Implemented temperature conversion logic in JavaScript.  
4. Added input validation to ensure only numeric values are accepted.  
5. Displayed results dynamically for Celsius, Fahrenheit, and Kelvin.  
6. Tested the application for different inputs and screen sizes.

## Outcome
- Successfully developed a fully functional temperature converter  
- Accurate conversion between Celsius, Fahrenheit, and Kelvin  
- User-friendly and visually appealing interface  
- Demonstrates understanding of DOM manipulation and basic JavaScript logic  
- Suitable beginner-level web development project
